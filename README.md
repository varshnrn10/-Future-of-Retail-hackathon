# -Future-of-Retail-hackathon


Our key goal is to improve space management at the point of sale by reducing the use of device setup for billing, which lowers operating costs and improves the retail store's financial performance. On converting the billing to a packing desk, the waiting time of customers is reduced which enhances the whole customer experience
